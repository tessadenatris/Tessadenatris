Hier heb ik aan mee geholpen met ontwerpen en maken


Korte stage - Madoo


In mijn tweede jaar mocht ik stage lopen, ik heb dit gedaan bij webdevelopment bedrijf Madoo. Ik kon hier aan het werk als stagiair webdevelopment / UX design.

Ik heb hier veel ervaring opgedaan over Wordpress websites. Ik kreeg al snel de verantwoordelijkheid om dingen aan te passen op echte websites en uiteindelijk mocht ik voor twee opdrachtgevers zelf de website ontwerpen en maken.

Een paar taken die ik heb uitgevoerd:
- Madoo eigen website door kijken en verbeteren
- Aanpassingen doorvoeren op verschillende websites
- Ontwerp voor een website maken
- Mobile first ontwerpen
- Teksten schrijven


Wat heb ik geleerd?

Ik heb veel geleerd over Wordpress, websites ontwerpen en over klant contact. Ik heb geleerd hoe je een website ontwerp vanaf niks tot het eindresultaat. Omdat de stage in corona tijd plaatsvond heb ik merendeel vanuit thuis moeten doen. Dit heeft mij geleerd om remote te werken en veel het initiatief te nemen als ik vragen had of toe was aan een nieuwe opdracht.
