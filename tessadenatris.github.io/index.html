Hi, ik ben Tessa de Natris, student CMD


De studie Communication and Multimedia Design (CMD) is een ontwerpopleiding voor digital interactive design op hbo- bachelorniveau. In mijn drie jaar heb ik ervaring opgedaan op het gebied van User Experience Design, Interaction Design, Front-end Development en Visual Interface Design.

Ik ben tijdens mijn studie vaardig geworden in programmeren (Html, CSS, Java). Ik kan goed overweg met programma’s als Adobe (Illustrator, InDesign, XD), Figma en Sketch. Mijn passie ligt bij User Experience design.

Voordat ik begon aan deze opleiding heb ik mijn MAVO en vervolgens HAVO examen gehaald. Na mijn HAVO examen heb ik een tussenjaar genomen en ben ik voor een halfjaar naar Australië gegaan om te reizen. Ik ben een enthousiast, onderzoekend en creatief. Ik vind het leuk om in een team samen te werken maar kan ook goed individueel werken.

Een beleving creëren of verbeteren waar mensen blij van worden is de reden dat ik mijn studie heb gekozen.
